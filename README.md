# Igreja Metodista · Loja de Uniformes

Site de vendas dos uniformes do time da Igreja Metodista (Uniforme Equipe e Camisa Torcida).

## Status atual

- `index.html` — página de login e cadastro (entrada do site).
- `dashboard.html` — página temporária pós-login, aguardando as próximas telas.

Backend: Supabase (autenticação por e-mail/senha + banco de dados com Row Level Security).
